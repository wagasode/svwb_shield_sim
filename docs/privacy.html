1. 本サイトでの情報収集

  • 本サイトは、名前・メールアドレス等の個人情報入力フォームを提供していません。
  • シミュレーション設定や結果は、ブラウザ上でのみ処理され、運営者サーバーへ送信しません。
  • 本サイト独自のアクセス解析タグや広告タグは埋め込んでいません。

2. GitHub Pages について

本サイトは GitHub Pages 上で配信されています。GitHub 側の基盤ログ等については GitHub のポリシーが適用されます。

  • What is GitHub Pages?
  • GitHub General Privacy Statement

3. 外部リンク先

カード名リンクなどで遷移する外部サイトでは、遷移先サイトのポリシーが適用されます。

4. 連絡

本ポリシーに関する問い合わせは、本リポジトリの Issue で受け付けます。
